### 2. weibo爬虫
这是一个微博爬虫程序。

**配置说明（获取 Cookie）：**

为了让爬虫正常工作，您需要获取微博的登录凭证（Cookie）。请按照以下步骤操作：

1.  **登录微博**：在浏览器中访问微博手机版网址 [https://m.weibo.cn](https://m.weibo.cn) 并完成登录。
2.  **打开开发者工具**：按下键盘上的 `F12` 键（或右键点击页面选择“检查”）。
3.  **定位 Application 面板**：
    *   在开发者工具顶部的标签栏中找到 **Application**（应用）。
    *   *注意：如果标签栏太窄没显示，请点击右侧的 `>>` 小箭头图标展开菜单查找。*
4.  **查找 Cookie**：
    *   在左侧侧边栏中找到 **Cookies** 选项，点击箭头展开。
    *   点击选中 `https://m.weibo.cn`。
5.  **获取 SUB 值**：
    *   在右侧出现的详细列表中，找到 **Name** 这一列叫 `SUB` 的那一行。
    *   双击该行 **Value** 对应的值（通常是一串以 `_2A` 开头的长字符）。
    *   复制这串字符，并将其粘贴到代码中的对应位置。
    *   *提示：如果列表中找不到 SUB，请尝试刷新页面或重新登录。*
