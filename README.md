Just do it
1.“index.html”是我的个人简介网站
2.“weibo爬虫”是微博爬虫程序，需要登录微博（手机版网址），按 F12 打开开发者工具。在顶部那一排标签里，找到 「应用」 (英文叫 Application)。如果没有，点一下那个 >> 小箭头就能看到。在左侧栏里找到 「Cookie」，点一下前面的小箭头展开，点击 https://m.weibo.cn。在右边的列表里，找到 名字(Name) 叫 SUB 的那一栏。如果找不到，请刷新页面或者重新登录。双击 SUB 对应的那一串 值(Value)（通常是以 _2A 开头的很长一串），复制它，填入代码中对应位置。
